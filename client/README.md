## Develop

### Prerequisite

We use [pnpm](https://pnpm.io/installation) as the projects package manager. Make sure to install it

```bash
npm install -g pnpm
```

### Run locally

```bash
pnpm install
pnpm start
```

## Linting and testing

```bash
pnpm lint
pnpm test
```


## Productions

### Build

```bash
pnpm build
```
